我的毕业设计

现代智慧农场的设计与实现
======
摘 要
------ 本毕业设计基于传感器技术、Zigbee 技术、计算机网络技术设计并实现了现代智 慧农场系统。系统由下位机信息采集和上位机信息处理两大部分构成。 下位机由传感器模块，数据处理模块，数据传输模块等构成。上位机由服务器， 安卓客户端以及 H5 界面组成。共同实现农场环境数据的采集、存储，显示、控制和 视频监测等功能。基本过程为传感器采集信息，STM32 单片机处理底层数据，通过 Zigbee 节点发送到协调器网关；网关通过 WIFI 模块与云端服务器建立 MQTT 网络链 接，将多个传感器数据发送到物联网云平台进行存储，在安卓端和 PC 端实时访问农 场传感器数据和对农场现场进行控制。 通过该系统，可以实时监测和控制农场环境变化，从而有效的控制植物生长，达 到科学管理的效果。

关键词： 传感器 无线网络 远程控制 智慧农场

Abstract
------

This graduation project is based on sensor technology, Zigbee technology and
computer network technology to design and implement the modern intelligent farm system.
The system consists of two parts: information acquisition of lower computer and
information processing of host computer.
The slave computer is composed of sensor module, data processing module, data
transmission module and so on. The upper computer is composed of server, Android client
and H5 interface. The functions of farm environment data acquisition, storage, display,
control and video monitoring are realized together. The basic process collects information
for the sensor, STM32 SCM processes the underlying data and sends it to the coordinator
gateway through the Zigbee node. The gateway establishes a MQTT network link through
the WIFI module and the cloud server. It sends multiple sensor data to the cloud platform
of the Internet of things to store and access the farm in real time in Android and H5 clients.
Sensor data and control of farm site.
The system can monitor and control the change of farm environment in real time, thus
effectively controlling plant growth and achieving scientific management effect.

Key words: Sensor wireless network remote control intelligent farm
